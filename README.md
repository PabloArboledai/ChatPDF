# ChatPDF

A project for interacting with PDF documents using AI-powered chat capabilities.

## Project Status

For detailed project status and updates, see [STATUS.md](STATUS.md).

## Overview

This repository is being set up to create a ChatPDF application that will allow users to:
- Upload and process PDF documents
- Ask questions about PDF content
- Get AI-powered responses based on document context
- Delegate tasks to cloud-based agents

## Current State

The project is in early development with configuration and setup in progress.
